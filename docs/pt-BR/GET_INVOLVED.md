# Como Participar

Você pode participar da SaaSCoop de várias formas.

## Contribuir tecnicamente

- Código
- Testes
- Revisão de Pull Requests
- Arquitetura
- Segurança

## Contribuir com documentação

- READMEs
- Tutoriais
- Traduções
- Guias
- Roadmaps

## Contribuir com comunidade

- Discussões
- Divulgação
- Organização de ideias
- Priorização comunitária
- Apoio a novos participantes

## Apoiar financeiramente

A SaaSCoop pretende estruturar um modelo de apoio comunitário transparente, preferencialmente por meio de uma plataforma como Open Collective.
