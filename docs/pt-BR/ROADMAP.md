# Roadmap da SaaSCoop

## 0–6 meses

- Consolidar a organização no GitHub
- Documentar governança
- Padronizar repositórios
- Transferir FreireAI e EmpathyAI
- Preparar candidatura ao Open Collective
- Iniciar comunidade de contribuidores

## 6–18 meses

- Ampliar projetos open source
- Publicar conteúdos educacionais
- Validar modelo de financiamento comunitário
- Evoluir governança participativa
- Avaliar estrutura jurídica adequada no Brasil

## 18–36 meses

- Consolidar operação sustentável
- Fortalecer comunidade internacional
- Avaliar formalização cooperativa
- Publicar relatórios de impacto
- Evoluir soluções SaaS e IA como bens comuns
