# Get Involved

You can participate in SaaSCoop in many ways.

## Technical contribution

- Code
- Testing
- Pull Request review
- Architecture
- Security

## Documentation contribution

- READMEs
- Tutorials
- Translations
- Guides
- Roadmaps

## Community contribution

- Discussions
- Promotion
- Idea organization
- Community prioritization
- Onboarding new participants

## Financial support

SaaSCoop intends to structure a transparent community funding model, preferably through a platform such as Open Collective.
