# Gobernanza

SaaSCoop adopta un enfoque de gobernanza transparente, participativo y orientado a la comunidad.

## Roles

- Apoyador
- Miembro de la comunidad
- Contribuidor
- Mantenedor
- Futuro cooperado

## Decisiones

Las decisiones deben documentarse preferiblemente mediante:

- Issues
- Discussions
- Pull Requests
- ADRs
- Roadmaps públicos

## Cuidados

SaaSCoop evita conflictos de interés, no opera con el sector público y no se presenta como cooperativa formalmente constituida antes de la validación jurídica adecuada.
