# Roadmap de SaaSCoop

## 0–6 meses

- Consolidar la organización en GitHub
- Documentar la gobernanza
- Estandarizar repositorios
- Transferir FreireAI y EmpathyAI
- Preparar la solicitud para Open Collective
- Iniciar la comunidad de contribuidores

## 6–18 meses

- Ampliar proyectos open source
- Publicar contenidos educativos
- Validar el modelo de financiación comunitaria
- Evolucionar la gobernanza participativa
- Evaluar la estructura jurídica adecuada en Brasil

## 18–36 meses

- Consolidar una operación sostenible
- Fortalecer la comunidad internacional
- Evaluar la formalización cooperativa
- Publicar informes de impacto
- Evolucionar soluciones SaaS e IA como bienes comunes
