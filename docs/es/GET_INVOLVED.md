# Cómo Participar

Puedes participar en SaaSCoop de varias formas.

## Contribución técnica

- Código
- Pruebas
- Revisión de Pull Requests
- Arquitectura
- Seguridad

## Contribución con documentación

- READMEs
- Tutoriales
- Traducciones
- Guías
- Roadmaps

## Contribución comunitaria

- Discusiones
- Difusión
- Organización de ideas
- Priorización comunitaria
- Apoyo a nuevos participantes

## Apoyo financiero

SaaSCoop pretende estructurar un modelo transparente de financiación comunitaria, preferiblemente mediante una plataforma como Open Collective.
