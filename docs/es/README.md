# SaaSCoop

**Tecnología como Bien Común.**

SaaSCoop es una iniciativa open source orientada al cooperativismo, dedicada a construir software, inteligencia artificial y educación digital de forma abierta, sostenible y comunitaria.

## Misión

Desarrollar soluciones digitales y de inteligencia artificial de manera cooperativa, transparente y sostenible, promoviendo soberanía tecnológica, trabajo humano digno y acceso democrático a la tecnología como bien común.

## Lo que estamos construyendo

- Soluciones open source
- Inteligencia artificial sostenible
- Educación digital abierta
- Herramientas para comunidades
- Gobernanza transparente
- Modelos de financiación comunitaria

## Proyectos principales

- FreireAI
- EmpathyAI
- SaaSCoop Education
- GitHub Governance
- SaaSCoop Standards

## Estado

SaaSCoop se encuentra actualmente en fase de estructuración y aún no se presenta aquí como una cooperativa formalmente constituida.
