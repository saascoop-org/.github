# Code of Conduct

## Our commitment

SaaSCoop is committed to providing a respectful, inclusive and collaborative environment for everyone.

We welcome participation regardless of background, experience level, language, location, disability, neurodiversity, gender, age, culture or technical skill level.

## Expected behavior

Participants are expected to:

- act respectfully
- communicate constructively
- assume good faith
- welcome newcomers
- respect different perspectives
- give and receive feedback responsibly
- focus criticism on ideas, not people

## Unacceptable behavior

Unacceptable behavior includes:

- harassment
- discrimination
- personal attacks
- intimidation
- doxxing
- abusive language
- repeated disruption of discussions
- publishing private information without consent

## Scope

This Code of Conduct applies to:

- GitHub repositories
- issues
- pull requests
- discussions
- community spaces
- events
- official communication channels

## Enforcement

Maintainers may take action when behavior violates this Code of Conduct.

Possible actions include:

- warning
- comment moderation
- temporary restriction
- removal from community spaces
- blocking from repositories

## Reporting

If you experience or witness unacceptable behavior, please contact the maintainers through the official communication channels listed in the repository or organization profile.

Reports should be handled respectfully and confidentially.

## Review

This Code of Conduct may be reviewed as SaaSCoop grows and its governance structure matures.
