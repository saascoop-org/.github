# SaaSCoop Roadmap

## Purpose

This roadmap describes the initial priorities for SaaSCoop as an open source cooperative initiative.

The roadmap may evolve through community participation, technical validation and legal/accounting guidance.

## Phase 1 — Institutional foundation

Goal: organize the public identity and governance of SaaSCoop.

Tasks:

- create organization profile
- document governance
- define contribution guidelines
- define code of conduct
- document security policy
- organize repositories
- standardize licenses
- create public roadmap

## Phase 2 — Project consolidation

Goal: consolidate existing open source initiatives under the SaaSCoop ecosystem.

Candidate projects:

- FreireAI
- EmpathyAI
- SaaSCoop Education
- AI Engineering Mentor
- SSIS to Fabric Migration Assistant
- SaaSCoop Governance

Tasks:

- review repository names
- preserve commit history
- update README files
- add licenses
- add contribution guides
- enable issues and discussions
- define maintainers

## Phase 3 — Community development

Goal: create a participatory community around SaaSCoop.

Tasks:

- publish onboarding materials
- create discussion spaces
- define community roles
- publish educational content
- collect feedback
- document decision-making
- invite contributors

## Phase 4 — Funding validation

Goal: validate a transparent and sustainable funding model.

Tasks:

- prepare Open Collective application
- document funding purpose
- define financial transparency practices
- define expense policy
- define project allocation criteria
- validate legal and accounting implications

## Phase 5 — Legal and cooperative validation

Goal: validate the most appropriate legal structure in Brazil.

Options to evaluate:

- cooperative
- association
- company
- hybrid structure
- fiscal sponsorship during early stage

Validation should consider:

- Brazilian cooperative law
- tax/accounting implications
- conflict of interest restrictions
- remote/digital operation
- community funding model
- remuneration model

## Phase 6 — Sustainable operation

Goal: evolve SaaSCoop from initiative to sustainable organization.

Tasks:

- formalize governance
- define membership rules
- define contributor remuneration model
- document financial reports
- publish impact reports
- maintain public roadmap
- grow the open source ecosystem
