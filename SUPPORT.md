# Support

## Community support

SaaSCoop is an open source initiative in structuring phase.

At this stage, support is provided through public community channels whenever possible.

## Where to ask for help

You may use:

- GitHub Issues for bugs and specific repository problems
- GitHub Discussions for ideas, questions and community conversations
- official SaaSCoop channels listed in the organization profile

## What to include

When asking for help, please include:

- repository name
- what you are trying to do
- what happened
- what you expected
- screenshots or logs, when useful
- your environment, when relevant

## No guaranteed support

SaaSCoop does not currently provide guaranteed response times.

Maintainers and contributors may respond according to availability.

## Commercial or institutional support

SaaSCoop is not currently offering formal commercial support through this GitHub organization.

Any future support model should be documented transparently and validated according to the legal and governance structure of the initiative.
